HD Wallet (WIP)

A minimal hierarchical deterministic (HD) wallet implementation built from scratch.

The goal is to understand and implement an HD wallet from first principles while depending on as few external libraries as practical (relying primarily on hashing libraries), while also serving as a learning resource for those interested in how HD wallets work. Once completed a full write up will be available.

Status

⚠️ Work in Progress

* Do not use in production.
* Do not store real funds with this wallet.
* The implementation has not been fully reviewed, tested, or audited.
